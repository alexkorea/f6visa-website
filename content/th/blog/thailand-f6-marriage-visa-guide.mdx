<h2 id="section-1">การจดทะเบียนสมรสที่อำเภอ (ไทย)</h2>

ในประเทศไทย การสมรสที่มีผลทางกฎหมายต้องจดทะเบียนที่ **อำเภอหรือสำนักงานเขต** การจัดงานแต่งงานตามพิธีศาสนาหรือจารีตประเพณีเพียงอย่างเดียวไม่มีผลทางกฎหมายตามกฎหมายไทย ทั้งสองฝ่ายต้องมาปรากฏตัวต่อเจ้าพนักงานจดทะเบียน

**ขั้นตอนการจดทะเบียนสมรสในไทย:**
1. ทั้งชาวเกาหลีและชาวไทยต้องมาที่อำเภอด้วยตนเอง
2. ฝ่ายเกาหลีต้องนำ **หนังสือรับรองความเป็นโสด** พร้อมการรับรองที่เหมาะสม
3. กรอกและยื่นแบบฟอร์มจดทะเบียนสมรสที่อำเภอ
4. รับ **ทะเบียนสมรสไทย (ทะเบียนสมรส)**

**หนังสือรับรองความเป็นโสดสำหรับชาวเกาหลี:**
- รับหนังสือรับรองความสัมพันธ์การสมรส (ยืนยันสถานะโสด) จากสำนักงานเขตหรือ Government24
- ประทับตรา Apostille หรือรับรองกงสุลที่สถานทูตเกาหลีในไทย
- อาจต้องแปลเป็นภาษาไทยและรับรองขึ้นอยู่กับข้อกำหนดของอำเภอ

> **เคล็ดลับ:** ข้อกำหนดแตกต่างกันตามอำเภอ ติดต่อสำนักงานจดทะเบียนท้องถิ่นล่วงหน้า หรือปรึกษาสถานทูตเกาหลีในไทย

---

<h2 id="section-2">ขั้นตอนการจดทะเบียนสมรสในเกาหลี</h2>

หลังจากการสมรสในไทยมีผลทางกฎหมายแล้ว คุณต้องจดทะเบียนสมรสในเกาหลีด้วยเพื่อให้ได้รับการยอมรับตามกฎหมายเกาหลี

**ขั้นตอนจดทะเบียนสมรสในเกาหลี:**
1. เตรียมทะเบียนสมรสไทย (ต้นฉบับ + การแปลภาษาเกาหลี + รับรอง)
2. เตรียมเอกสารทะเบียนครอบครัวของคู่สมรสชาวเกาหลี
3. นำสำเนาหนังสือเดินทางคู่สมรสชาวไทย
4. ไปที่สำนักงานเขต (주민센터) หรือยื่นทางไปรษณีย์
5. ระยะเวลาดำเนินการ: โดยทั่วไป 1–3 วันทำการ
6. รับใบสำคัญการสมรสเกาหลี (혼인관계증명서)

> **หมายเหตุ:** การแปลภาษาเกาหลีและรับรองทะเบียนสมรสไทยเป็นสิ่งบังคับ การแปลต้องทำโดยนักแปลที่ได้รับการรับรอง; การรับรองต้องทำที่สำนักงานรับรองเอกสารของเกาหลี

---

<h2 id="section-3">รายการเอกสารสำหรับวีซ่า F-6</h2>

วีซ่า F-6-1 (การย้ายถิ่นฐานเพื่อการสมรส) โดยทั่วไปยื่นโดยคู่สมรสชาวไทยที่สถานทูตหรือสถานกงสุลใหญ่เกาหลีในไทย

| เอกสาร | หน่วยงานออก | หมายเหตุ |
|---|---|---|
| แบบฟอร์มขอวีซ่า | แบบฟอร์มของสถานกงสุลเกาหลี | ที่สถานกงสุลหรือออนไลน์ |
| หนังสือเดินทางคู่สมรสชาวไทย (ต้นฉบับ) | ส่วนตัว | อายุเหลืออย่างน้อย 6 เดือน |
| ทะเบียนสมรสไทย (ต้นฉบับ + การแปลภาษาเกาหลี + รับรอง) | อำเภอไทย | ต้องใช้นักแปลที่ได้รับการรับรอง |
| ใบสำคัญการสมรสเกาหลี | สำนักงานเขตเกาหลี / Government24 | ออกภายใน 3 เดือน |
| สำเนาหนังสือเดินทางผู้รับรองชาวเกาหลี | ส่วนตัว | |
| เอกสารทะเบียนครอบครัวผู้รับรองชาวเกาหลี | สำนักงานเขตเกาหลี / Government24 | ออกภายใน 3 เดือน |
| เอกสารพิสูจน์รายได้ผู้รับรอง | สำนักงานภาษี / นายจ้าง | ใบรับรองรายได้ ฯลฯ |
| หลักฐานความสัมพันธ์แท้จริง | เตรียมเอง | รูปภาพ ประวัติการสนทนา ประวัติการเยี่ยม |
| บัตรประจำตัวประชาชนไทย (บัตรประชาชน) | ส่วนตัว | |

> **เคล็ดลับ:** รายการเอกสารอาจเปลี่ยนแปลง ตรวจสอบ [เว็บไซต์สถานทูตเกาหลีในไทย](https://overseas.mofa.go.kr/th-ko/index.do) ก่อนยื่น

<h2 id="section-4">การแปลและรับรองทะเบียนสมรสไทย</h2>

ทะเบียนสมรสไทย (ทะเบียนสมรส) เขียนเป็นภาษาไทย สำหรับการจดทะเบียนสมรสในเกาหลีและการขอวีซ่า F-6 ต้องใช้ **การแปลภาษาเกาหลี** โดยนักแปลที่ได้รับการรับรอง และรับรองที่สำนักงานรับรองเอกสารของเกาหลี

**ขั้นตอน:**
1. ว่าจ้างนักแปลที่ได้รับการรับรองแปลจากภาษาไทยเป็นเกาหลี (ต้องมีตราประทับและลายเซ็นนักแปล)
2. ไปที่สำนักงานรับรองเอกสารของศาลเกาหลีหรือทนายความรับรองเอกสารเพื่อรับรอง
3. ยื่นต้นฉบับและเอกสารแปลพร้อมกันเพื่อตรวจสอบความถูกต้อง

> **คำเตือน:** การแปลโดยสมาชิกครอบครัวหรือผู้ที่ไม่ได้รับการรับรองจะไม่ได้รับการยอมรับเป็นเอกสารทางการ

---

<h2 id="section-5">ขั้นตอนการตรวจสอบเอกสาร</h2>

หน่วยงานตรวจคนเข้าเมืองเกาหลีและสถานกงสุลอาจต้องการตรวจสอบความถูกต้องของทะเบียนสมรสไทยเพื่อป้องกันการยื่นเอกสารปลอม

**วิธีการตรวจสอบ:**
- **Apostille:** ไทยเป็นสมาชิกอนุสัญญาเฮก ดังนั้นเอกสารไทยสามารถประทับตรา apostille ได้ที่กระทรวงการต่างประเทศไทย เอกสารที่มี apostille สามารถยื่นต่อสถานกงสุลเกาหลีได้โดยไม่ต้องรับรองกงสุลเพิ่มเติม
- **การรับรองกงสุล:** สถานทูตเกาหลีในไทยสามารถให้การรับรองกงสุลเป็นทางเลือก

> **เคล็ดลับ:** ข้อกำหนดแตกต่างกันตามสถานกงสุล ยืนยันวิธีการที่ต้องการก่อนดำเนินการเสมอ

<h2 id="section-6">ระยะเวลาดำเนินการและค่าใช้จ่าย</h2>

| ขั้นตอน | เวลาโดยประมาณ | หมายเหตุ |
|---|---|---|
| จดทะเบียนสมรสในไทย | 1–3 วัน (ในไทย) | เวลาเตรียมเอกสารแยกต่างหาก |
| จดทะเบียนสมรสในเกาหลี | 1–5 วันทำการ | รวมการแปล/รับรอง: 1–2 สัปดาห์ |
| การพิจารณาวีซ่า F-6 | โดยทั่วไป 4–8 สัปดาห์ | แตกต่างตามสถานกงสุลและช่วงเวลา |
| การขอเอกสารเพิ่มเติม | ตามสถานการณ์ | หากมีการขอ จะใช้เวลาเพิ่ม |

**รายการค่าใช้จ่ายหลัก:**
- ค่าธรรมเนียมจดทะเบียนสมรสไทย
- ค่าแปลเอกสาร
- ค่ารับรองเอกสาร
- ค่า apostille หรือการรับรองกงสุล
- ค่าธรรมเนียมขอวีซ่า F-6
- ค่าบริการตัวแทน (ถ้ามี)

<h2 id="section-7">สาเหตุทั่วไปที่วีซ่า F-6 ถูกปฏิเสธ</h2>

**1. หลักฐานความสัมพันธ์แท้จริงไม่เพียงพอ**
- ระยะเวลาความสัมพันธ์สั้นหรือขาดบันทึกการพบปะ
- อุปสรรคด้านภาษาที่รุนแรง
- ขาดประวัติการสนทนา รูปภาพ หรือประวัติการเยี่ยม

**2. ไม่ผ่านข้อกำหนดด้านรายได้**
- รายได้ผู้รับรองชาวเกาหลีต่ำกว่าเกณฑ์ของกระทรวงยุติธรรม
- เอกสารพิสูจน์รายได้ไม่เพียงพอสำหรับฟรีแลนซ์หรือผู้ประกอบอาชีพอิสระ

**3. ข้อผิดพลาดหรือขาดเอกสาร**
- ข้อผิดพลาดในการแปล การรับรองขาดหาย เอกสารหมดอายุ

**4. การละเมิดกฎตรวจคนเข้าเมืองก่อนหน้า**
- ชาวไทยมีประวัติอยู่เกินกำหนดหรือทำงานโดยไม่ได้รับอนุญาตในเกาหลี

**5. ปัญหาทางกฎหมายเกี่ยวกับการสมรสในไทย**
- ไม่ได้ดำเนินการจดทะเบียนที่อำเภอหรือเอกสารไม่ตรงกัน

> **เคล็ดลับ:** หากวีซ่าถูกปฏิเสธ ยังสามารถยื่นใหม่ได้ ระบุสาเหตุการปฏิเสธที่ชัดเจนและเตรียมเอกสารเพิ่มเติมที่ครอบคลุม การทำงานร่วมกับผู้เชี่ยวชาญด้านการตรวจคนเข้าเมืองเพื่อยื่นใหม่เป็นสิ่งที่แนะนำอย่างยิ่ง

<h2 id="section-8">คำถามที่พบบ่อย (FAQ)</h2>

**Q1. จัดงานแต่งงานในไทยโดยไม่จดทะเบียนที่อำเภอ สามารถขอ F-6 ได้หรือไม่?**
ไม่ได้ วีซ่า F-6 ต้องการการสมรสที่มีผลทางกฎหมาย ต้องจดทะเบียนที่อำเภอไทยและในเกาหลีก่อนยื่นขอ

**Q2. คู่สมรสชาวไทยเดินทางเข้าเกาหลีด้วยวีซ่าท่องเที่ยว (C-3) แล้วเปลี่ยนเป็น F-6 ในประเทศได้หรือไม่?**
โดยหลักการ วีซ่า F-6 ควรยื่นที่สถานกงสุลเกาหลีในต่างประเทศ การเปลี่ยนสถานะในประเทศอาจเป็นไปได้ภายใต้เงื่อนไขบางอย่าง แต่ไม่ได้ถ้าอยู่ในสถานะเกินกำหนด ปรึกษาผู้เชี่ยวชาญก่อนเสมอ

**Q3. ชาวเกาหลีที่ไม่รู้ภาษาไทยสามารถจดทะเบียนสมรสในไทยได้หรือไม่?**
ได้ ด้วยความช่วยเหลือจากล่ามท้องถิ่นหรือที่ปรึกษาการตรวจคนเข้าเมือง ทั้งสองฝ่ายต้องมาที่อำเภอและเอกสารต้องเป็นภาษาไทย

**Q4. คู่สมรสชาวไทยที่เคยแต่งงานมาก่อนต้องใช้เอกสารเพิ่มเติมหรือไม่?**
ต้องมีหลักฐานการหย่าร้างหรือใบมรณะบัตร (ใบหย่าหรือใบมรณะบัตรไทย) พร้อมการแปลภาษาเกาหลีและรับรอง

**Q5. มีการสัมภาษณ์ระหว่างการพิจารณาวีซ่า F-6 หรือไม่?**
สถานกงสุลบางแห่งจัดการสัมภาษณ์เพื่อตรวจสอบความแท้จริงของการสมรส ทั้งสองคู่สมรสควรเตรียมคำตอบที่สอดคล้องกัน แนะนำให้มีคำแนะนำจากผู้เชี่ยวชาญในการเตรียมตัว
