# รายการเอกสารวีซ่าแต่งงาน F-6 เกาหลี 2026 ครบถ้วน

เอกสารที่จำเป็นสำหรับวีซ่าตั้งถิ่นฐานโดยการแต่งงาน F-6 แตกต่างกันตามประเภท ตามมาตรา 9-2 ของกฎกระทรวงกฎหมายคนเข้าเมืองและมาตรฐานรายละเอียดสำหรับเอกสารขอวีซ่า นี่คือรายการตรวจสอบเอกสารปี 2026

## 1. ภาพรวมประเภทวีซ่า F-6

วีซ่าตั้งถิ่นฐานโดยการแต่งงาน F-6 แบ่งออกเป็นสามประเภทย่อย:

- **F-6-1**: คู่สมรสของพลเมืองเกาหลี
- **F-6-2**: บิดาหรือมารดาที่เลี้ยงดูบุตรที่เกิดจากการสมรสกับพลเมืองเกาหลี ตามที่รัฐมนตรีว่าการกระทรวงยุติธรรมรับรอง
- **F-6-3**: บุคคลที่เลี้ยงดูบุตรผู้เยาว์ในประเทศหลังหย่าร้างหรือสูญเสียคู่สมรสที่เป็นพลเมืองเกาหลี

## 2. เอกสารบังคับร่วม (ทุกประเภท)

เอกสารต่อไปนี้จำเป็นสำหรับวีซ่า F-6 ทุกประเภท:

- [ ] **แบบฟอร์มขอวีซ่า** (รูปแบบกระทรวงยุติธรรม)
- [ ] **หนังสือเดินทาง** (มีอายุอย่างน้อย 6 เดือน รวมถึงสำเนา)
- [ ] **รูปถ่ายมาตรฐาน** (3.5 ซม. × 4.5 ซม. ถ่ายภายใน 6 เดือนที่ผ่านมา)
- [ ] **ใบเสร็จชำระค่าธรรมเนียม**

## 3. เอกสารบังคับสำหรับ F-6-1 (คู่สมรสพลเมืองเกาหลี)

F-6-1 เป็นประเภทที่พบบ่อยที่สุด สำหรับชาวต่างชาติที่แต่งงานอย่างถูกกฎหมายกับพลเมืองเกาหลี

**เอกสารของผู้ยื่นคำร้อง:**
- [ ] ใบสำคัญการสมรส (ออกภายใน 3 เดือน พร้อม Apostille หรือการรับรองกงสุล)
- [ ] รูปถ่ายงานแต่งงาน (2–3 รูปจากพิธี)
- [ ] ใบรับรองความสัมพันธ์ในครอบครัว (ถ้ามี)
- [ ] หนังสือรับรองความประพฤติ (ออกโดยตำรวจในประเทศผู้ยื่น พร้อม Apostille)

**เอกสารของผู้สนับสนุน (คู่สมรส):**
- [ ] จดหมายเชิญ (รูปแบบกระทรวงยุติธรรม)
- [ ] หนังสือค้ำประกันส่วนบุคคล (รูปแบบกระทรวงยุติธรรม)
- [ ] ทะเบียนบ้าน (ออกภายใน 3 เดือน)
- [ ] ใบรับรองความสัมพันธ์ในครอบครัว (ออกภายใน 3 เดือน)
- [ ] เอกสารยืนยันรายได้ (ใบรับรองหักภาษี ณ ที่จ่าย, ยืนยันการชำระเบี้ยประกันสุขภาพ ฯลฯ)
- [ ] หนังสือรับรองการจ้างงานหรือใบรับรองการจดทะเบียนธุรกิจ
- [ ] ใบสำคัญการสมรส

> **หมายเหตุ**: อาจต้องใช้เอกสารเพิ่มเติมขึ้นอยู่กับว่าได้เสร็จสิ้น International Marriage Guidance Program (การศึกษาก่อน) หรือไม่ สำหรับบางประเทศ (เวียดนาม จีน ฟิลิปปินส์ กัมพูชา มองโกเลีย อุซเบกิสถาน ฯลฯ) การศึกษาก่อนเป็นข้อบังคับ

## 4. เอกสารบังคับสำหรับ F-6-2 (บิดา/มารดาเลี้ยงดูบุตร)

ใช้บังคับเมื่อเลี้ยงดูบุตรที่เกิดจากการสมรสกับพลเมืองเกาหลี

- [ ] ใบสูติบัตรของบุตรหรือใบรับรองความสัมพันธ์ในครอบครัว
- [ ] เอกสารยืนยันการเลี้ยงดูบุตร (ทะเบียนบ้านรวมถึงบุตร, บันทึกการชำระค่าเลี้ยงดู ฯลฯ)
- [ ] เอกสารการหย่าร้างหรือสูญเสีย (คำพิพากษาหย่าร้าง, ใบมรณบัตร ฯลฯ)
- [ ] ทะเบียนบ้าน (รวมถึงบุตร)

## 5. เอกสารบังคับสำหรับ F-6-3 (เลี้ยงดูบุตรหลังหย่าร้าง/สูญเสีย)

ใช้บังคับเมื่อเลี้ยงดูบุตรผู้เยาว์ในประเทศหลังหย่าร้างหรือสูญเสียคู่สมรสที่เป็นพลเมืองเกาหลี

- [ ] คำพิพากษาหย่าร้างหรือการยืนยันการหย่าร้างโดยสมัครใจ / ใบมรณบัตร
- [ ] เอกสารสิทธิ์การดูแลบุตร (คำสั่งศาลหรือข้อตกลง)
- [ ] ใบรับรองความสัมพันธ์ในครอบครัวของบุตร
- [ ] เอกสารยืนยันสถานะการเลี้ยงดูบุตรปัจจุบัน (ใบรับรองการลงทะเบียนโรงเรียน, เอกสารประกันสุขภาพ ฯลฯ)

## 6. ข้อควรสังเกตสำคัญในการเตรียมเอกสาร

- **การแปลและการรับรอง**: เอกสารภาษาต่างประเทศต้องแนบการแปลรับรองโดยโนตารีเป็นภาษาเกาหลี
- **Apostille หรือการรับรองกงสุล**: หากประเทศเป็นสมาชิกอนุสัญญาเฮก ต้องใช้ Apostille; สำหรับประเทศที่ไม่ใช่สมาชิก ต้องมีการรับรองกงสุลจากสถานทูตหรือสถานกงสุลเกาหลีในประเทศนั้น
- **ระยะเวลาความถูกต้อง**: ใบรับรองส่วนใหญ่ต้องออกภายใน 3 เดือน
- **เอกสารต้นฉบับ**: เมื่อส่งสำเนา ต้องนำต้นฉบับมาเพื่อการยืนยัน

## คำถามที่พบบ่อย

**ถ. ฉันสามารถยื่นขอได้หรือไม่หากยังไม่ได้จดทะเบียนสมรส?**
ต. วีซ่า F-6-1 ต้องการการสมรสที่ถูกกฎหมายในทั้งสองประเทศ ต้องดำเนินการจดทะเบียนสมรสในเกาหลีหรือการลงทะเบียนอย่างเป็นทางการในอีกประเทศหนึ่งให้เสร็จสมบูรณ์ก่อนยื่นขอ

**ถ. จะเกิดอะไรขึ้นหากเอกสารไม่ครบถ้วน?**
ต. หากเอกสารไม่ครบถ้วน การตรวจสอบอาจถูกระงับหรือคุณอาจถูกขอให้ยื่นเอกสารเพิ่มเติม เนื่องจากบางสถานทูตหรือสถานกงสุลให้โอกาสเพียงครั้งเดียว จึงเป็นสิ่งสำคัญที่จะต้องเตรียมเอกสารให้ครบถ้วนตั้งแต่การยื่นครั้งแรก

**ถ. ฉันสามารถขอหนังสือรับรองความประพฤติได้ที่ไหน?**
ต. ออกโดยหน่วยงานตำรวจหรือหน่วยงานรัฐบาลที่เกี่ยวข้องในประเทศที่คุณอาศัยอยู่ และต้องมีการรับรอง Apostille ตามด้วยการแปลรับรอง ในเกาหลี สามารถใช้บริการสอบถามประวัติอาชญากรรมของสำนักงานตำรวจแห่งชาติ (Minwon 24) ได้
